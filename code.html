Comparação de Linguagens de Programação

Linguagem Paradigma(s) Tipagem Popularidade Facilidade de Aprendizado Ecossistema/Comunidade Aplicações Notáveis Comentários
JavaScript Multiparadigma (procedural, orientada a objetos, funcional) Fraca Muito alta Moderada Vasta Desenvolvimento web, Node.js, front-end Conheço há um tempo por ser bastante famosa, ainda não mexi com ela. Parece ser fácil de usar.
Java Orientada a Objetos Forte Alta Moderada Grande Desenvolvimento de aplicativos, servidores Muitos falam mal, então tenho um pouco de receio. Penso que deve ser difícil. Vi pouco sobre e irei estudar sobre para a residência desse semestre.
PHP Multiparadigma (procedural, orientada a objetos) Fraca Moderada Fácil Moderada Desenvolvimento web Vi um pouco dessa linguagem quando meu grupo integrou front com um banco de dados e utilizamos PHP para isso. Achei relativamente fácil.
Python Multiparadigma (orientada a objetos, funcional) Dinâmica Muito alta Fácil Grande Ciência de dados, automação, web A famosinha da parada. Vários livos sobre, inclusive sobre seu uso em análise de dados. É realmente bem enxuta e fácil de usar.
C# Orientada a Objetos Forte Alta Moderada Grande Desenvolvimento de aplicativos Windows Sei que algumas empresas a usam. Não utilizei ainda, mas estudei C semestre passado.
Go Estática (Compilada) Forte Moderada Moderada Moderada Desenvolvimento de sistemas, servidores Vi essa linguagem sendo usada em um curso de Pentest. Não usei ainda.
